# Установка

AudioPie доступен для Linux, Windows и macOS. Выберите подходящий способ установки ниже.

## Скачать готовые сборки

Самый простой способ установить AudioPie — скачать готовые сборки со страницы [GitHub Releases](https://github.com/ikloster03/audiopie/releases).

### Linux

1. Скачайте файл `.AppImage` со страницы релизов
2. Сделайте его исполняемым:
   ```bash
   chmod +x AudioPie_*.AppImage
   ```
3. Запустите приложение:
   ```bash
   ./AudioPie_*.AppImage
   ```

Альтернативно, можно распаковать архив `linux-unpacked` и запустить исполняемый файл `audiopie` напрямую.

### Windows

1. Скачайте установщик `.exe` со страницы релизов
2. Запустите установщик и следуйте инструкциям
3. Запустите AudioPie из меню Пуск

### macOS

1. Скачайте файл `.dmg` со страницы релизов
2. Откройте DMG и перетащите AudioPie в папку Программы
3. Запустите AudioPie из Launchpad или Spotlight

## Сборка из исходников

Если вы предпочитаете собрать AudioPie из исходного кода, выполните следующие шаги:

### Необходимые компоненты

- [Node.js](https://nodejs.org/) версии 18 или выше
- [npm](https://www.npmjs.com/) (поставляется с Node.js)
- [Git](https://git-scm.com/)

### Шаги

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ikloster03/audiopie.git
   cd audiopie
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите сервер разработки (опционально):
   ```bash
   npm run dev
   ```

4. Соберите для продакшена:
   ```bash
   npm run build
   ```

Собранное приложение будет в директории `release/{version}`.

## FFmpeg

AudioPie использует FFmpeg для обработки аудио. Приложение включает встроенную версию FFmpeg, поэтому вам не нужно устанавливать её отдельно.

Однако, если вы хотите использовать системную установку FFmpeg:

1. Откройте AudioPie
2. Перейдите в **Настройки** (значок шестерёнки)
3. Укажите путь к исполняемым файлам FFmpeg и FFprobe

Подробности смотрите в [руководстве по настройке FFmpeg](./configuration/ffmpeg).

## Устранение неполадок

### Приложение не запускается

- **Linux**: Убедитесь, что AppImage имеет права на выполнение
- **macOS**: Возможно, нужно разрешить приложение в Системные настройки → Защита и безопасность
- **Windows**: Если Windows Defender блокирует приложение, нажмите "Подробнее" → "Всё равно выполнить"

### Ошибки FFmpeg

Если вы видите ошибки, связанные с FFmpeg:

1. Проверьте путь к FFmpeg в настройках
2. Попробуйте использовать встроенный FFmpeg (оставьте пути пустыми в настройках)
3. Убедитесь, что FFmpeg имеет права доступа к вашим аудиофайлам

### Другие проблемы

При возникновении других проблем:

1. Проверьте [FAQ](./faq)
2. Поищите в [GitHub Issues](https://github.com/ikloster03/audiopie/issues)
3. Создайте новый issue с подробной информацией о проблеме
