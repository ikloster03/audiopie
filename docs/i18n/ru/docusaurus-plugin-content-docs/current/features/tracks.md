# Работа с треками

Список треков — основа вашего проекта аудиокниги. Это руководство охватывает всё, что нужно знать об управлении аудиотреками в AudioPie.

## Добавление треков

### Перетаскивание

Самый быстрый способ добавить треки — перетаскивание:

1. Откройте файловый менеджер
2. Выберите один или несколько MP3-файлов
3. Перетащите их на список треков AudioPie
4. Отпустите для добавления в проект

### Диалог выбора файлов

Альтернативно, используйте диалог выбора файлов:

1. Нажмите кнопку **+ Добавить треки** в заголовке
2. Перейдите к вашим аудиофайлам
3. Выберите один или несколько файлов (удерживайте `Ctrl` или `Shift` для множественного выбора)
4. Нажмите **Открыть**

:::tip
AudioPie в настоящее время поддерживает только **MP3** файлы. Другие форматы (WAV, FLAC, OGG) будут добавлены в будущих версиях.
:::

## Возможности списка треков

### Информация о треке

Каждый трек отображает:
- **Номер трека** — Позиция в списке
- **Название** — Отображаемое имя (из имени файла или ID3-тегов)
- **Длительность** — Продолжительность в формате ЧЧ:ММ:СС

### Переупорядочивание треков

Треки можно переупорядочить перетаскиванием:

1. Нажмите и удерживайте трек
2. Перетащите его на нужную позицию
3. Отпустите

Номера треков автоматически обновятся.

### Удаление треков

Чтобы удалить трек:

1. Наведите курсор на трек
2. Нажмите кнопку **×** (удалить) справа
3. Трек удаляется немедленно

:::caution
Удаление трека также удаляет связанную с ним главу. Это действие нельзя отменить.
:::

## Связь треков и глав

При добавлении треков AudioPie автоматически создаёт главы на основе информации о треках:

| Свойство трека | Свойство главы |
|----------------|----------------|
| Имя файла | Название главы |
| Длительность | Время окончания главы |
| Порядок | Порядок главы |

Если вы переупорядочите треки, связанные главы также переупорядочатся.

## Лучшие практики

### Именование файлов

Для лучших результатов называйте MP3-файлы описательно:

```
Хорошо:
├── 01 - Введение.mp3
├── 02 - Глава 1 - Начало.mp3
├── 03 - Глава 2 - Путешествие.mp3
└── 04 - Эпилог.mp3

Избегайте:
├── track001.mp3
├── track002.mp3
├── audio_20231215_recording.mp3
└── final_v2_FIXED.mp3
```

### Качество аудио

- Используйте одинаковый битрейт для всех треков
- Рекомендуется: 128 кбит/с или 64 кбит/с для голоса
- Убедитесь, что все треки имеют одинаковую частоту дискретизации

### Порядок треков

- Добавляйте треки в правильном порядке, или
- Используйте перетаскивание для переупорядочивания после добавления
- Числовые префиксы (01, 02, 03) помогают поддерживать порядок при импорте

## Устранение неполадок

### Трек не добавляется

Если трек не добавляется:
- Убедитесь, что это валидный MP3-файл
- Проверьте права доступа к файлу
- Попробуйте перекодировать файл другим инструментом

### Длительность показывается как 0:00

Если длительность отображается неправильно:
- Файл может быть повреждён
- Попробуйте перекодировать файл
- Проверьте установку FFmpeg в настройках

### Неправильный порядок воспроизведения

Если главы появляются не в том порядке:
- Переупорядочьте треки в списке треков
- Главы обновятся автоматически
