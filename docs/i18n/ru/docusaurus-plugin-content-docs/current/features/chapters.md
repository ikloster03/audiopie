# Редактирование глав

Главы — это то, что делает аудиокниги особенными. Они позволяют слушателям переходить непосредственно к определённым частям книги. AudioPie предоставляет мощные инструменты редактирования глав.

## Понимание глав

Каждая глава в AudioPie имеет:

| Свойство | Описание |
|----------|----------|
| **Название** | Название главы, отображаемое в плеерах |
| **Время начала** | Когда начинается глава (рассчитывается автоматически) |
| **Время окончания** | Когда заканчивается глава (рассчитывается автоматически) |

Главы последовательны и непрерывны — конец одной главы является началом следующей.

## Автоматическое создание глав

При добавлении треков AudioPie автоматически создаёт главы:

```
Трек: 01 - Введение.mp3 (5:30)
  → Глава: "01 - Введение" (0:00 - 5:30)

Трек: 02 - Глава первая.mp3 (15:00)
  → Глава: "02 - Глава первая" (5:30 - 20:30)

Трек: 03 - Глава вторая.mp3 (12:30)
  → Глава: "03 - Глава вторая" (20:30 - 33:00)
```

## Редактирование названий глав

### Быстрое редактирование

1. Перейдите на вкладку **Главы**
2. Нажмите на название любой главы
3. Введите новое название
4. Нажмите `Enter` или кликните вне поля для сохранения

### Пакетное переименование

Для единообразного именования рассмотрите шаблон:

- `Глава 1: Начало`
- `Глава 2: Первые шаги`
- `Пролог`, `Эпилог`, `Послесловие`

## Переупорядочивание глав

:::warning
Переупорядочивание глав НЕ переупорядочивает аудио. Последовательность аудио определяется списком треков. Переупорядочивание глав влияет только на маркеры глав.
:::

Чтобы изменить порядок и аудио, и глав:
1. Перейдите на панель **Треки**
2. Переупорядочьте треки (перетаскиванием)
3. Главы автоматически обновятся

## Тайминг глав

Время глав рассчитывается на основе суммарной длительности треков:

```
Глава 1: 0:00:00 - 0:05:30  (длительность Трека 1)
Глава 2: 0:05:30 - 0:20:30  (Трек 1 + Трек 2)
Глава 3: 0:20:30 - 0:33:00  (Трек 1 + Трек 2 + Трек 3)
```

Время отображается в формате `ЧЧ:ММ:СС` для удобства.

## Расширенные операции с главами

### Объединение глав

Хотя AudioPie напрямую не объединяет главы, вы можете достичь этого:

1. Переименовав главы для одинаковой концептуальной группировки
2. Или объединив исходные MP3-файлы перед импортом

### Пользовательские точки глав

В настоящее время главы соответствуют трекам 1:1. Для пользовательских точек глав внутри одного трека:

1. Используйте аудиоредактор для разделения трека
2. Импортируйте полученные файлы как отдельные треки
3. Назовите их соответствующим образом

### Очень длинные треки

Если у вас один длинный аудиофайл:

1. Рассмотрите его разделение на логические главы
2. Используйте инструменты типа `mp3splt` или Audacity
3. Импортируйте полученные файлы

## Лучшие практики

### Именование глав

Хорошие названия глав:
- ✅ Описательные и значимые
- ✅ Единообразные по формату
- ✅ Не слишком длинные (50 символов или меньше)

Избегайте:
- ❌ Общих названий типа "Трек 1", "Часть А"
- ❌ Имён файлов с расширениями
- ❌ Очень длинных описаний

### Количество глав

- Короткие книги: 5-15 глав
- Средние книги: 15-30 глав
- Длинные книги: 30-50+ глав

Слишком много очень коротких глав может раздражать слушателей.

## Как плееры отображают главы

Разные плееры аудиокниг отображают главы по-разному:

| Плеер | Отображение глав |
|-------|------------------|
| Apple Books | Список глав, маркеры на полосе прокрутки |
| Audiobookshelf | Список глав с таймингом |
| Smart AudioBook Player | Выпадающий список глав |
| Prologue | Список глав, касание для перехода |

## Устранение неполадок

### Главы не соответствуют аудио

- Убедитесь, что треки в правильном порядке
- Проверьте, правильно ли читается длительность треков
- Пересоберите проект

### Время глав неправильное

- Проверьте, нет ли у какого-то трека неправильной длительности
- Попробуйте удалить и снова добавить проблемные треки
- Убедитесь, что FFmpeg работает корректно (Настройки)

### Главы не отображаются в плеере

Некоторым плеерам нужно:
- Пересканировать библиотеку
- Повторно импортировать аудиокнигу
- Обновить свою базу данных
