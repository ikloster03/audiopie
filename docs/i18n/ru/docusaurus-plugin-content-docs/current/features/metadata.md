# Управление метаданными

Метаданные делают вашу аудиокнигу обнаруживаемой и хорошо организованной. AudioPie поддерживает поля метаданных, которые встраиваются в финальный файл M4B.

## Поля метаданных

### Обязательные

| Поле | Описание |
|------|----------|
| **Название** | Название аудиокниги. Обязательно для сохранения и сборки. |

### Рекомендуемые

| Поле | Описание |
|------|----------|
| **Автор** | Автор книги (писатель) |
| **Жанр** | Жанр или категория книги |
| **Обложка** | Изображение обложки аудиокниги |

### Опциональные

| Поле | Описание |
|------|----------|
| **Описание** | Синопсис или описание книги |

## Редактирование метаданных

1. Переключитесь на вкладку **Метаданные** в правой панели
2. Заполните нужные поля
3. Метаданные автоматически сохраняются с проектом

### Название

Название — самое важное поле:

```
Хорошие названия:
- "Великий Гэтсби"
- "Гарри Поттер и философский камень"
- "Проект «Аве Мария»"

Избегайте:
- "Аудиокнига"
- "Без названия"
- "Книга 1"
```

### Автор

Поле автора указывает, кто написал книгу:

```
Автор: Энди Вейер
```

Для нескольких авторов разделяйте запятыми:

```
Авторы: Джеймс Кори, Тай Фрэнк
```

### Жанр

Используйте стандартные категории жанров для лучшей организации:

```
Хорошие жанры:
- Научная фантастика
- Фэнтези
- Детектив
- Биография
- Саморазвитие
```

## Обложка

### Добавление обложки

Есть два способа добавить обложку:

1. **Нажатие для выбора** — Нажмите на область обложки или кнопку "Выбрать обложку"
2. **Перетаскивание** — Перетащите файл изображения прямо на область обложки

Поддерживаемые форматы: JPG, PNG, WebP

### Требования к обложке

Для лучшей совместимости:

| Свойство | Рекомендация |
|----------|--------------|
| Формат | JPG или PNG |
| Размер | 500×500 — 1400×1400 пикселей |
| Соотношение сторон | Квадратное (1:1) |
| Размер файла | Менее 500 КБ |

:::tip Где найти обложки
Вы можете найти обложки аудиокниг на:
- [Audible](https://audible.com)
- [Goodreads](https://goodreads.com)
- [Open Library](https://openlibrary.org)
- [ЛитРес](https://litres.ru)
:::

### Обложка в плеерах

Обложка появляется в:
- Виде библиотеки плеера
- Экране воспроизведения
- Экране блокировки (мобильные)
- Области уведомлений

## Описание

Поле описания поддерживает длинный текст. Используйте его для:

```
От автора «Марсианина» — новый научно-фантастический
триллер об астронавте, который просыпается на космическом
корабле без памяти о том, как он туда попал и какова его миссия.

«Незабываемая история выживания и научной изобретательности.»
```

## Как хранятся метаданные

AudioPie встраивает метаданные, используя стандартные теги M4B/M4A:

| Поле | Тег |
|------|-----|
| Название | `©nam` |
| Автор | `©ART` |
| Жанр | `©gen` |
| Описание | `desc` |
| Обложка | `covr` |

Эти теги читаются всеми основными плеерами аудиокниг.

## Метаданные из исходных файлов

Если ваши MP3-файлы имеют ID3-теги, AudioPie использует их для:
- Отображаемых имён треков
- Начальных названий глав

Однако метаданные на уровне книги (название, автор и т.д.) нужно вводить вручную.

## Лучшие практики

### Будьте последовательны

Если вы создаёте несколько аудиокниг:
- Используйте единые соглашения об именовании
- Форматируйте имена авторов одинаково
- Используйте те же категории жанров

### Держите в чистоте

- Удаляйте лишние пробелы
- Исправляйте регистр букв
- Используйте правильные имена (не псевдонимы)

## Устранение неполадок

### Метаданные не отображаются в плеере

- Некоторые плееры кэшируют метаданные — попробуйте пересканировать
- Проверьте, поддерживает ли плеер метаданные M4B
- Попробуйте повторно импортировать аудиокнигу

### Обложка не отображается

- Убедитесь, что изображение не слишком большое (>2 МБ)
- Попробуйте другой формат изображения (JPG вместо PNG)
- Проверьте размеры изображения (должно быть квадратным)

### Специальные символы в названии

Большинство специальных символов работают нормально. При проблемах:
- Избегайте эмодзи в названии
- Используйте стандартные кавычки и тире
- Удалите необычные символы Unicode
