# Быстрый старт

Это руководство проведёт вас через создание первой аудиокниги с AudioPie за несколько минут.

## Шаг 1: Запустите AudioPie

После [установки](./installation) AudioPie, запустите приложение. Вы увидите экран Менеджера проектов.

Здесь вы можете:
- **Новый проект** — Начать новый проект аудиокниги
- **Открыть проект** — Открыть существующий файл `.audiopie`

Нажмите **Новый проект**, чтобы начать.

## Шаг 2: Добавьте аудиотреки

После создания нового проекта вы увидите основное рабочее пространство с пустым списком треков.

### Добавление треков

Есть два способа добавить MP3-треки:

1. **Перетаскивание** — Просто перетащите MP3-файлы из файлового менеджера на список треков
2. **Кнопка Добавить треки** — Нажмите кнопку **+ Добавить треки** в заголовке и выберите файлы в диалоге

Ваши треки появятся в левой панели, а главы будут автоматически созданы на основе имён файлов.

## Шаг 3: Организуйте главы

Переключитесь на вкладку **Главы** в правой панели для управления структурой глав аудиокниги.

### Операции с главами

- **Переименование** — Нажмите на название главы, чтобы отредактировать его
- **Переупорядочивание** — Перетащите главы для изменения порядка
- **Объединение** — Выберите несколько глав и объедините их в одну
- **Разделение** — Разделите длинную главу на более короткие части

Главы автоматически рассчитываются по времени на основе ваших аудиотреков.

## Шаг 4: Заполните метаданные

Переключитесь на вкладку **Основное**, чтобы добавить информацию об аудиокниге:

| Поле | Описание |
|------|----------|
| **Название** | Название аудиокниги (обязательно) |
| **Автор** | Автор книги |
| **Жанр** | Жанр/категория книги |
| **Описание** | Синопсис или описание книги |
| **Обложка** | Изображение обложки (нажмите для выбора или перетащите) |

:::tip
Поле Название обязательно для сохранения проектов и сборки аудиокниг.
:::

## Шаг 5: Соберите аудиокнигу

Когда вы довольны главами и метаданными, пришло время собрать файл M4B.

1. Нажмите кнопку **Собрать** (значок молотка) в заголовке
2. Выберите, куда сохранить выходной файл
3. Дождитесь завершения процесса кодирования

Диалог прогресса покажет текущий статус:

- **Анализ** — Чтение информации об аудиофайлах
- **Кодирование** — Конвертация аудио в формат AAC
- **Добавление глав** — Встраивание маркеров глав
- **Финализация** — Запись метаданных и обложки

## Шаг 6: Наслаждайтесь аудиокнигой!

После завершения ваш файл M4B готов к использованию с любым плеером аудиокниг:

- **Apple Books** — Встроенная поддержка в macOS/iOS
- **Audiobookshelf** — Сервер аудиокниг с открытым исходным кодом
- **Smart AudioBook Player** — Популярный плеер для Android
- **Prologue** — Плеер аудиокниг для iOS

## Сохранение проекта

Не забудьте сохранить проект! Нажмите **Сохранить** (или `Ctrl+S`), чтобы сохранить работу как файл `.audiopie`. Это позволяет:

- Продолжить редактирование позже
- Пересобрать с другими настройками
- Сохранить редактирование глав и метаданные
