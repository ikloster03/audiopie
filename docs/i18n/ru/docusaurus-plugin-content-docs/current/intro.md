# Добро пожаловать в AudioPie

**AudioPie** — это настольное приложение для создания аудиокниг в формате M4B из MP3-файлов с возможностью редактирования глав.

## Что такое AudioPie?

AudioPie — это мощный и простой в использовании инструмент, который позволяет:

- 🎵 **Конвертировать MP3 в M4B** — Превратите коллекцию MP3-файлов в единую аудиокнигу M4B
- 📖 **Редактировать главы** — Создавайте, переименовывайте, переупорядочивайте и объединяйте главы перетаскиванием
- 🎨 **Управлять метаданными** — Редактируйте название книги, автора, жанр, описание и обложку
- 💾 **Сохранять проекты** — Сохраняйте работу и продолжайте позже
- 🌍 **Многоязычность** — Доступен на русском и английском языках

## Почему формат M4B?

Формат M4B — золотой стандарт для аудиокниг, потому что он:

- ✅ Поддерживает навигацию по главам
- ✅ Запоминает позицию воспроизведения
- ✅ Работает со всеми основными плеерами (Apple Books, Audiobookshelf и др.)
- ✅ Обеспечивает лучшее сжатие, чем MP3
- ✅ Поддерживает обложки и богатые метаданные

## Системные требования

- **Операционная система**: Linux, Windows или macOS
- **Node.js**: Версия 18 или выше (для разработки)
- **FFmpeg**: Требуется для обработки аудио (включён в приложение)
